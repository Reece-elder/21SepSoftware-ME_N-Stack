<old_name>.copyDatabase('old_name','new_name')
old_name.dropDatabase

Delete
use DB_NAME
db.dropDatabase();


Collection Commands

Once you've specified use <db_name> you can pass in db instead of the database name

Create - Creates a collection

db.createCollection('Lizards');

Read - Show collections
show collections
db.show collections;

Delete 
db.<collection_name>.drop();

Update
db.collection_name.renameCollection('new_name')
